Project Eridan - FPS Shooter Game

Kill all the white enemies! Use crates to hide behind them but remember that they are destructable.

This FPS shooter game is created as a spin-off from the Three.js Journey by Bruno Simon.

Recomendations

Use PC or laptop because it is very difficult to make controls work with touch screens.
